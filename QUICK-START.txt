🎯 الخطوات السريعة (5 خطوات فقط!)
══════════════════════════════════════════════════════════════

┌─────────────────────────────────────────────────────────────┐
│ 1️⃣  سجل الدخول على GoDaddy                                 │
│    → https://www.godaddy.com                                │
│    → My Products → Web Hosting → cPanel                     │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 2️⃣  افتح File Manager                                       │
│    → انتقل إلى مجلد: public_html                            │
│    → احذف أي ملفات قديمة (إذا وجدت)                        │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 3️⃣  ارفع الملفات                                           │
│    → اختر: Upload                                           │
│    → ارفع **كل محتويات** مجلد dist                          │
│    → (ليس المجلد نفسه، فقط الملفات داخله!)                 │
│    → تأكد: index.html موجود مباشرة في public_html          │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 4️⃣  أضف ملف .htaccess                                       │
│    → أنشئ ملف جديد في public_html                           │
│    → الاسم: .htaccess                                       │
│    → الصق المحتوى من ملف .htaccess في المشروع              │
│    → احفظ                                                   │
└─────────────────────────────────────────────────────────────┘

┌─────────────────────────────────────────────────────────────┐
│ 5️⃣  فعّل SSL                                               │
│    → cPanel → SSL/TLS Status                                │
│    → فعّل للنطاق                                            │
│    → انتظر 5-10 دقائق                                       │
└─────────────────────────────────────────────────────────────┘
